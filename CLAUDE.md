# ACW Carrosserie — Maquette One-Page

## Description
Maquette de redesign du site acw-carrosserie.fr — Carrossier artisan à Gérardmer (Vosges)

## Client
- **Nom :** ACW Carrosserie (Alexis WISSEMBERG)
- **Secteur :** Carrosserie automobile
- **Localisation :** 10 impasse des Granitiers, 88400 Gérardmer
- **URL originale :** https://www.acw-carrosserie.fr/

## Stack technique
- HTML/CSS/JS inline (fichier unique)
- Google Fonts (Bebas Neue + Source Sans 3)
- Vanilla JS (Intersection Observer, custom cursor)

## Design
- **Style UI :** Exaggerated Minimalism (ui-ux-pro-max)
- **Typographie :** Bold Statement — Bebas Neue (display) + Source Sans 3 (body)
- **Palette :** Rouge ACW #E30613, Noir #111111, Off-white #F5F5F0
- **Layout :** Split-screen hero, bento grid services, zigzag about, horizontal values

## Éléments Awwwards
1. Custom cursor interactif (dot + ring)
2. Typographie expressive (Bebas Neue massive clamp 3.5rem-7rem)
3. Scroll-driven clip-path reveals
4. Counter animations (chiffres qui montent)
5. Parallaxe subtile sur galerie
6. Marquee/ticker décoratif
7. Animations de hover variées (scale, color invert, slide overlays)
